nnAudio.librosa_functions.hz_to_mel¶

nnAudio.librosa_functions.hz_to_mel(frequencies, htk=False)¶

Convert Hz to Mels .. rubric:: Examples

>>> librosa.hz_to_mel(60)
0.9
>>> librosa.hz_to_mel([110, 220, 440])
array([ 1.65,  3.3 ,  6.6 ])
Parameters

  • frequencies (number or np.ndarray [shape=(n,)] , float) – scalar or array of frequencies

  • htk (bool) – use HTK formula instead of Slaney

Returns

mels – input frequencies in Mels

Return type

number or np.ndarray [shape=(n,)]
